# /godmode:research

User research skill — persona creation, survey design, interview scripts, journey mapping, pain point identification, competitive UX analysis, usability test planning, and Jobs-to-be-Done framework.

## Usage

```
/godmode:research                           # Full user research workflow
/godmode:research --persona                 # Create user personas
/godmode:research --journey                 # Map user journey
/godmode:research --survey                  # Design survey questions
/godmode:research --interview               # Generate interview scripts
/godmode:research --competitive             # Competitive UX analysis
/godmode:research --jtbd                    # Jobs-to-be-Done analysis
/godmode:research --usability               # Plan usability tests
```

## What It Does

1. Synthesizes user data into actionable personas
2. Maps user journeys with touchpoints and pain points
3. Designs surveys with proper question methodology
4. Creates interview scripts with probing follow-ups
5. Analyzes competitor UX strengths and weaknesses
6. Applies JTBD framework to uncover real user needs

## Output
- Research artifacts at `docs/research/<topic>/`
- Personas, journey maps, and analysis reports
- Commit: `"research: <topic> — user research artifacts"`
